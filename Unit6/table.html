The HTML Document Object Model (DOM) is a standard for accessing, adding, deleting, or updating elements of an HTML document. JavaScript has been designed around the idea of the DOM with a hierarchy of parent and child objects. In this example, ahierarchy of objects is defined using the document.createElement() method with various table objects as arguments. The following (child, parent) relationships are defined among various objects: (table, body), (tblbody, table), (row, tblbody), (cell,row), and (cellText, cell).
